Event QR Code FAQs
Which calendar apps does it work with?
The iCal format (VCALENDAR) is supported by Google Calendar, Apple Calendar, Microsoft Outlook, Samsung Calendar, and virtually all calendar apps on Android and iPhone.

Wedding Invitation QR Code Kaise Banaye

Shaadi ke invitation card pe event QR code add karo. Mehman scan karein aur event turant unke Google Calendar ya iPhone Calendar mein save ho jaaye. Date, time, venue — sab automatically save hoga. No typing, no confusion.

Conference and Seminar QR Code

Event organizers can add calendar QR codes to registration confirmation emails, badges, and venue signage. Attendees scan once and the event saves to all their devices that sync with Google Calendar or Outlook. Reduces no-shows by keeping the event top-of-mind.

iCal QR Code — Works on All Devices

QRMaker generates standard iCal format (VCALENDAR/VEVENT) which is the universal calendar standard. It works on Google Calendar, Apple Calendar (iPhone and Mac), Microsoft Outlook, Samsung Calendar, and virtually every calendar app on any platform worldwide.

Can I add a venue address to the calendar event?
Yes. Enter the full venue address in the location field. When saved to the calendar, it appears as a tappable link that opens Google Maps or Apple Maps for directions.
Can I set a reminder in the QR code event?
The iCal standard supports reminders, but QRMaker creates the event without a default reminder since reminder preferences vary by device. The user can add a reminder after the event is saved to their calendar.
Will the event include a reminder?
The QR code creates the event without a reminder by default. After adding, users can set their own reminder in their calendar app.
Can I add an online meeting link?
Yes — paste the Zoom/Google Meet/Teams link in the Location or Description field. It will be saved with the calendar event.
